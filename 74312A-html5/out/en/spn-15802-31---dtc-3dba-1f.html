SPN-15802-31 / DTC-3DBA-1F

DEF Pressure Line Heater Failure and DEF Tank Temperature is Too Low to Perform Afterrun

Document ID: 2470 | Version: 3.2 | Last Modified Date: Oct 24, 2024

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Fault Description: 

The electronic control unit (ECU) monitors the temperature of the DEF in the tank and the pressure to the high pressure DEF hose during afterrun. If the ECU determines that the DEF pressure line is not behaving normally during the afterrun cycle, after sufficient time has passed for the system to unfreeze and for normal flow to occur, this fault occurs.

Cause: 

The ECU has determined that a mechanical failure or problem exists in the heated high pressure DEF hose.

Possible Failure Modes:

  1. Faulty DEF supply module connector, blocked or leaking.

  2. Faulty DEF dosing module connector, blocked or leaking.

  3. Faulty high pressure heated DEF line, blocked or leaking.

  4. Faulty DEF line electrically heated circuit.

Suggested Spare Parts:

  • Heated DEF lines

  • DEF supply module

  • DEF dosing module

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Inspect the high pressure DEF line for blockage or damage to the hose and O-rings.

    1. If blockage is found, and if blockage is due to freezing, continue to the next step.

    2. If blockage or damage is found, and blockage is not due to freezing, repair or replace as required. Use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset and return the machine to service.

    3. If no blockage or damage is found, continue to Step 4.

  3. Check the hose heating element and connectors for any damage or internal failure. Check for supply voltage, ground from machine harness side connector, and resistance on hose side connector.

    1. If damage is found, repair or replace as required. Use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset and return the machine to service.

    2. If no damage is found, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn

  4. Visually inspect the DEF supply module connector for damage, blockage, or signs of leakage.

    1. If any damage, blockage, or signs of leakage are found, repair or replace as required. Use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset and return the machine to service.

    2. If no damage, blockage, or signs of leakage are found, continue to the next step.

  5. Visually inspect the DEF dosing module connector for damage, blockage, or signs of leakage.

    1. If any damage, blockage, or signs of leakage are found, repair or replace as required. Use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset and return the machine to service.

    2. If no damage, blockage, or signs of leakage are found, continue to the next step.

  6. Check the DEF dosing module functionality.

    • Remove the DEF dosing module from the DOC and inspect the DEF dosing module for signs of crystallization. Perform a Urea Dosing Pump Check (UDPC) and verify the spray pattern. This will also clean the nozzle of any internal crystals.

    • Perform a Urea Dosing Leakage Check (UDLC).

    1. If no DEF is sprayed from the nozzle during the UDPC test, remove the hose from the DEF dosing module and perform the UDPC again to verify that the DEF supply module is pumping fluid correctly.

    2. If the DEF dosing module fails any of the tests, replace the DEF dosing module, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, and then return the machine to service.

    3. If the DEF dosing module passes all of the tests, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
