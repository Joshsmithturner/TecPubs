SPN-15534-31 / DTC-3CAE-1F

Downstream NOx Sensor Lambda Signal Deviation - NOx Sensor Possible Removal Detected

Document ID: 5781 | Version: 2.2 | Last Modified Date: Oct 24, 2024

Note

Because this fault causes inducement, it is necessary to perform the fault repair verification test of the Selective Catalytic Reduction (SCR) once the fault has been resolved.

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Fault Description: 

The NOx sensor downstream of the SCR has internal monitoring to optimize operational conditions and reports the status with the data measured. If the Lambda data values measured by the sensor are not within the minimum/maximum curves specified in the electronic control unit (ECU) according to the Lambda value, this fault occurs.

Cause: 

The ECU has determined that the downstream NOx sensor is not measuring the correct Lambda values.

Possible Failure Modes:

  1. Faulty SCR downstream NOx sensor, hardware or firmware.

Suggested Spare Parts:

  • NOx sensor

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Check the correct operation of the downstream NOx sensor, following the Test the NOx Sensor procedure.

    1. If the sensor is working correctly, contact the Tigercat Industries Service Department for more help.

    2. If the sensor is not working correctly, continue to the next step.

  3. Replace the downstream NOx sensor, use the PT-Box or LogOn to perform the Component Replacement > Replace NOx Downstream Sensor procedure, and then use the PT-Box or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
