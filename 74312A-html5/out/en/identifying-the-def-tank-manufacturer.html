Identifying the DEF Tank Manufacturer

Document ID: 7496 | Version: 2.10 | Last Modified Date: Sep 9, 2025

This machine may be equipped with either an FPT DEF tank, a KUS DEF tank (1st Gen), or a KUS DEF tank (2nd Gen). Check which tank is equipped before starting troubleshooting.

FPT DEF Tank

On FPT DEF tanks, the level and temperature sensors are integrated into the sending unit, which communicates with the ECU via signal cables. The DEF level and temperature information is sent to the ECU as raw voltage signals, which are then converted into physical temperature and level values by an analog-to-digital converter. The quality sensor is a separate sensor on the bottom of the tank, which communicates with the ECU via CAN.

KUS DEF Tank (1st Gen)

On KUS DEF tanks (1st Gen), the quality sensor is integrated into the sending unit along with the level and temperature sensors. All sensors communicate with the ECU via CAN through one MCU.

KUS DEF Tank (2nd Gen)

On KUS DEF tanks (2nd Gen), the level and temperature sensors are integrated into the sending unit, which communicates with the ECU via CAN. The quality sensor is a separate sensor on the bottom of the tank, which also communicates with the ECU via CAN.
