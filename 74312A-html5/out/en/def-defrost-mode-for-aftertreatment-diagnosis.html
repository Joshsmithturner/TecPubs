DEF Defrost Mode for Aftertreatment Diagnosis

Document ID: 2368 | Version: 3.5 | Last Modified Date: Oct 23, 2024

Note

Before starting work, read and follow all servicing safety precautions as listed in the machine and attachment operator and service manuals.

Description:

If performing aftertreatment tests with the PT-Box in cold weather and the temperature is below the required setting, an ADBLUE DEFROST error message will be displayed and prevent aftertreatment diagnosis tests from being performed.

Required Action:

Validate DEF tank temperature before starting any aftertreatment diagnosis tests.

DEF Defrost Mode Error

Procedure:

  1. Start the engine and validate the DEF tank temperature.

    In the MD3, select the J1939 Parameter Inputs screen and then scroll down to DEF Tank Temp PIN to view the current DEF temperature reading from the DEF tank.

    In the MD4, select the Aftertreatment measure group and then scroll down to DEF Tank Temp PIN to view the current DEF temperature reading from the DEF tank.

  2. A minimum DEF tank temperature of 22°C (72°F) is needed before performing any aftertreatment diagnosis. If the temperature is below 22°C (72°F) run the engine at high speed to warm the temperature to 22°C (72°F).

  3. If the machine is in derate, a counter reset will need to be done to enable the machine to run at high speed.

  4. Once the DEF tank temp is at required temperature the aftertreatment diagnosis tests can be performed. If the engine is turned off to run an aftertreatment test in cold weather the machine may have to be restarted and warmed up again for a few minutes to resume testing.

  5. Any active codes for the dosing module or supply module will also prevent aftertreatment diagnosis tests from being performed. All active codes need to be addressed and resolved before proceeding with diagnosis.
