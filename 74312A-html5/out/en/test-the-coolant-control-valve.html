Test the Coolant Control Valve

Document ID: 12912 | Version: 2.3 | Last Modified Date: Oct 28, 2024

Description:

The coolant control valve is a component of the aftertreatment system and has the dual purpose of preventing the DEF from freezing and also preventing the DEF dosing module from overheating. The switching valve is a 2 position/3-way solenoid valve. The first position allows cooling of the dosing module, while the second position allows heating of the DEF tank and relative pipes. Depending on the temperatures detected in the DEF tank and supply module, the coolant control valve will close or open the passage of the engine coolant in the DEF tank heating circuit. The cooling circuit of the dosing module is always open.

The DEF tank heating circuit is used in two different modes.

Defrost Mode

In defrost mode, the coolant circuit is used together with the electrically heated DEF lines to heat the DEF system. Defrost mode is activated at engine start when any of the following temperature conditions are met:

  • DEF tank temperature < −8°C (18°F)

  • DEF supply module temperature < −4°C (25°F)

  • Ambient temperature < −9°C (16°F)

Defrost mode continues to heat the DEF tank and supply module until the DEF tank and DEF supply module temperatures are above their respective thresholds, or until 68 minutes have passed.

Heating Mode

In heating mode, the coolant control valve and the electrically heated DEF lines are activated independently based on the temperature conditions. Heating mode is automatically activated after defrost to bring the temperatures up to operating conditions, and during normal operation if the temperatures fall.

Test:

Note

This procedure can only be performed in cool temperatures in which the DEF defrost or heating modes will be activated. If the coolant control valve needs to be tested in warmer ambient temperatures, contact the Tigercat Industries Service Department for more help.

  1. Verify that the DEF system temperatures are below the threshold for defrost or heating (DEF tank is below 17°C (63°F) or DEF supply module is below −2°C (28°F).

  2. START the engine and wait until the coolant has warmed up to 25°C (77°F).

  3. Check that there is coolant flowing out of the coolant control valve towards the DEF supply module.

    By hand, feel the temperature of the coolant control valve at the inlet side (from the engine) and compare it to the temperature of the outlet side (to the sending unit on the DEF tank).

    1. If there is no flow and the temperature on the outlet side does not feel the same as the inlet side, this indicates that the component may not be working correctly. Return to the troubleshooting procedure.

    2. If there is flow and the temperature feels the same on the inlet and outlet sides of the coolant control valve, continue to the next step.

  4. Use the PT-Box tool or LogOn to disable and clear the fault code. Restore the operating conditions of the vehicle and check that the fault code relating to the coolant control valve does not activate.

    1. If the fault code relating to the coolant control valve is not active, this indicates that the sensor is operating correctly. Return to the troubleshooting procedure.

    2. If the fault code relating to the coolant control valve is active, check the efficiency of the wiring and return to the troubleshooting procedure.
