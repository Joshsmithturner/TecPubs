SPN-5157-31 / DTC-1425-1F

Fuel Rail Pressure Sensor: Voltage is Too High

Document ID: 999 | Version: 4.3 | Last Modified Date: Sep 10, 2025

Figure 1. Fuel Rail Pressure Sensor Connector

Fault Description: 

The electronic control unit (ECU) monitors the fuel rail pressure sensor signal circuit. If the ECU determines that the voltage in the rail pressure sensor signal circuit is higher than expected, this fault occurs.

Cause: 

The ECU has detected a value greater than 4.9 V for a period greater than 0.2 seconds.

Possible Failure Modes:

  1. Faulty fuel rail pressure sensor reference voltage, excessive.

  2. Faulty fuel rail pressure sensor, internal failure.

  3. Faulty fuel rail pressure sensor signal circuit, short to high source or open circuit condition.

Suggested Spare Parts:

  • Fuel rail pressure sensor

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 6.

  2. Check the status of the following related faults:

    SPN-58413-31 / DTC-E42D-1F

    ECU Internal: Sensor 5-volt Supply 3

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  3. Check the fuel rail pressure sensor signal circuit for an open circuit condition:

    Disconnect the fuel rail pressure sensor.

    Disconnect the ECU connector-2 (engine harness connector).

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 36

    Fuel Rail Pressure Sensor (Plug) position 2

    There should be continuity.

    1. If there is no continuity, there is an open circuit condition in the fuel rail pressure sensor signal circuit. Locate and repair the broken conductor.

    2. If there is continuity, leave the connectors disconnected and continue to the next step.

  4. Check the fuel rail pressure sensor signal circuit for a short to high source condition:

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 36

    ECU Connector-2 (Plug) position 11

    There should be no continuity.

    ECU Connector-2 (Plug) position 36

    All other positions in ECU Connector-2 (Engine Harness Connector)

    There should be no continuity.

    1. If there is continuity, there is a short to high source condition in the fuel rail pressure sensor signal circuit. Locate and repair the shorted conductor.

    2. If there is no continuity, leave the connectors disconnected and continue to the next step.

  5. Check the fuel rail pressure sensor signal circuit for a short to key power condition:

    With the key in the RUN position, use a multimeter to perform the following voltage check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 36

    Chassis ground

    There should be no voltage.

    1. If there is voltage, there is a short to key power condition in the fuel rail pressure sensor signal circuit. Locate and repair the shorted conductor.

    2. If there is no voltage, continue to the next step.

  6. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  7. Replace the fuel rail pressure sensor and torque to 30–40 N-m (22–30 ft-lbs), then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
