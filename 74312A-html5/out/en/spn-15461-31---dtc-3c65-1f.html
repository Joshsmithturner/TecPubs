SPN-15461-31 / DTC-3C65-1F

High Pressure Pump Fuel Delivery Quantity at Low Speed is Too High

Fault Description: 

The fuel metering valve is a magnetic proportional valve (MPROP) which controls the desired fuel flow to the fuel rail. The fuel metering valve adjusts the quantity of fuel reaching the high-pressure pump based on commands received from the ECU. If the ECU determines that proper fuel pressure cannot be reached at engine idle speed, leakage in the system is assumed and this fault occurs.

Cause: 

The ECU has determined that commanded fuel rail pressure is lower than expected at engine idle speed.

Possible Failure Modes:

  1. Faulty fuel filter, clogged.

  2. Faulty low-pressure fuel lines, air entrance, clogged or damaged.

  3. Faulty fuel injectors, leaking.

  4. Faulty 5 bar relief valve, leaking.

  5. Faulty fuel metering valve (MPROP).

Suggested Spare Parts:

  • Fuel filters

  • Fuel injectors

  • 5 bar relief valve

  • Fuel metering valve (MPROP)

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Return the machine to service.

  2. Replace the fuel filters and fuel/water separator filter, then use the PT-Box tool or LogOn to check the status of the fault.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Return the machine to service.

  3. Check the low-pressure fuel lines for air entrance, clogging, or blockage. There should be no blockage or damage.

    1. If blockage or damage is found, repair or replace as needed, and use the PT-Box tool or LogOn to verify that the fault is no longer active.

    2. If no blockage or damage is found, continue to the next step.

  4. Check for leaking fuel injectors. With the fuel pressure leak kit, install the injector backflow line and route to an appropriate container, then bring the engine up to operating temperature. At low idle, injector backflow should be 80 ml per minute maximum. Refer to Fuel Injector Leakage Test for the full procedure.

    1. If injector backflow is greater than 80 ml per minute, use the fuel rail injector plug kit to isolate each injector, one at a time. Determine which injectors are faulty and replace them.

    2. If injector backflow is less than or equal to 80 ml per minute, continue to the next step.

  5. Remove the 5 bar relief valve, clean thoroughly, and reinstall. If the fault is not resolved, replace the 5 bar relief valve with a new unit and then use the PT-Box tool or LogOn to check the status of the fault again.

    Tighten the valve to 55 N-m (41 ft-lbs). Refer to the machine's Service Manual for more information.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  6. Replace the fuel metering valve (MPROP), use the PT-Box tool or LogOn to perform the Component Replacement > Replace Rail Metering Unit procedure, and then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
