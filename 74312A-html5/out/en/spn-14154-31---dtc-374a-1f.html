SPN-14154-31 / DTC-374A-1F

DEF Backflow Line Plausibility Error

Document ID: 6335 | Version: 2.3 | Last Modified Date: Jun 9, 2025

Note

Because this fault causes inducement, it is necessary to perform the fault repair verification test of the Selective Catalytic Reduction (SCR) once the fault has been resolved.

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Fault Description: 

To protect the DEF system against damages due to frozen DEF, the system must be evacuated at each system stop. First the system is depressurized by actuating the reverse valve (pump sucks DEF from feeding tube), the pump is set to 0.25%, and the dosing valve is fully closed until the DEF pressure drops below 250.00 hPa. Then the system is evacuated by still actuating the reverse valve with the pump set to 0.75% and the dosing valve open until the tubing is empty (remaining quantity 0% of total tubing volume). If the DEF pressure does not drop below 250.00 hPa within 40 seconds during the depressurization phase, the system is not OK and this fault occurs.

Cause: 

The electronic control unit (ECU) has determined that the DEF system was not evacuated properly, as system pressure has remained above 250.00 hPa for longer than 40 seconds.

Possible Failure Modes:

  1. Faulty DEF system pressure or backflow line (tube), restricted.

  2. Faulty DEF reverting valve, blocked (stuck) closed.

  3. Faulty DEF supply module pressure sensor, values too high.

Suggested Spare Parts:

  • DEF supply module

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 4.

  2. Check DEF supply module operation.

    Use the PT-Box tool or LogOn to perform the Urea Dosing System Test (UDST). Follow the UDST Troubleshooting Guideline to determine the problem.

    1. If the UDST identifies a problem, follow the UDST Troubleshooting Guideline to locate and repair the failure.

    2. If no problems are found, continue to the next step.

  3. Check for blockage or damage in the DEF supply module backflow line (tube).

    Visually and tactually inspect the DEF supply module backflow line for restrictions or damage. There should be no blockage or damage.

    Note

    Contamination in the backflow connector is difficult to see. The most effective way to rule out a blockage due to contamination is to replace the connector.

    1. If blockage or damage is found, repair or replace the line (tube).

    2. If no blockage or damage is found, continue to the next step.

  4. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  5. Replace the DEF supply module, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
