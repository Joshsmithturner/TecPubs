Check the ECU Power Supply

Document ID: 5247 | Version: 3.2 | Last Modified Date: Sep 10, 2025

Preparation:

  • Ignition switch set to the STOP position

Procedure:

  1. Check the ECU power supply for the correct voltage.

    Disconnect the ECU connector-1 (machine harness connector).

    With the key in the RUN position, use a multimeter to perform the following voltage check:

    From

    To

    Value

    ECU Connector-1 (Plug) position 1

    Chassis ground

    24 V

    ECU Connector-1 (Plug) position 25

    Chassis ground

    24 V

    ECU Connector-1 (Plug) position 26

    Chassis ground

    24 V

    ECU Connector-1 (Plug) position 49

    Chassis ground

    24 V

    ECU Connector-1 (Plug) position 73

    Chassis ground

    24 V

    1. If the voltage is not present for one or more of the checks, refer to the appropriate vehicle Service Manual and electrical schematics to locate and restore supply voltage to the ECU.

    2. If the voltage is present on all of the checks, continue to the next step.

  2. Check the ECU wiring harness for an open circuit condition.

    Disconnect the controller connectors and fuses or relays linked to the inspected terminals/cables.

    Use a multimeter to measure the resistance between the ECU connector-1 (machine harness connector) and the indicated destination point:

    From

    To

    Value

    ECU Connector-1 (Plug) position 1

    Battery + via 30A fuse and power junction distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 25

    Battery + via 30A fuse and power junction distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 26

    Battery + via 30A fuse and power junction distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 49

    Battery + via 30A fuse and power junction distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 73

    Battery + via 30A fuse and power junction distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 3

    Battery − via chassis ground distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 5

    Battery − via chassis ground distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 28

    Battery − via chassis ground distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 52

    Battery − via chassis ground distribution point (see machine schematic)

    0–0.2 Ω

    ECU Connector-1 (Plug) position 75

    Battery − via chassis ground distribution point (see machine schematic)

    0–0.2 Ω

    1. If one or more of the checks is not within the specified range, refer to the appropriate vehicle Service Manual and electrical schematics to locate and correct the open circuit condition.

    2. If all the values are within the specified range, continue to the next step.

  3. The ECU power supply works correctly.
