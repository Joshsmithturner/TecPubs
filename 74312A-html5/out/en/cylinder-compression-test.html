Cylinder Compression Test

Document ID: 4032 | Version: 2.5 | Last Modified Date: Oct 3, 2025

Required Tools:

  • Cylinder compression tester tool

  • Cylinder pressure loss tester tool

Preparation:

  • Engine oil level is correct

  • Engine is warm, if possible

Procedure:

Note

The compression test is used to compare the pressure conditions in the individual combustion chambers. A simplified cylinder compression test can also be run using LogOn. If using LogOn, follow the on-screen instructions to perform the test and then compare the results with the possible root causes listed in this procedure.

  1. Prepare the machine for a compression test.

    • Check that the battery voltage is sufficient.

    • Check that the vehicle fuses are OK.

    • Check that there are no deposits on the undersides of the valve heads.

    • Leave the key in the STOP position.

    • Remove the fuel injectors.

    • With suitable piping fixtures, seal off the fuel rail outlets to the fuel injectors.

    • Unplug the connectors of the crankshaft and camshaft speed sensors.

  2. Check cylinder compression.

    1. Follow the cylinder compression tester tool operator manual to connect the pressure gauge to cylinder number 1.

    2. Crank the engine for 8 seconds and record the maximum pressure reached.

    3. Repeat the compression test for each cylinder twice.

    4. Calculate the average compression pressure for each cylinder and compare the results.

      • Compression should build up quickly in a healthy engine.

      • There should be similar compression values between all cylinders.

    5. If the cylinder compression is NOT OK, investigate these possible root causes:

      Symptom

      Root Cause

      Low compression reading on the first stroke, followed by gradually increasing pressure on successive strokes

      • Worn piston rings

      • Worn cylinder

      Low compression reading on the first stroke, which does not build up during successive strokes

      • Leaking valves

      • Faulty head gasket

      Low reading from two adjacent cylinders

      • Faulty head gasket between the two cylinders

      Deviating compression in one or more cylinders

      • Engine mechanical failure

      Locate and correct the cause, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    6. If the cylinder compression is OK, continue to the next step.

  3. Prepare the machine for a cylinder pressure loss test.

    • Leave the key in the STOP position.

    • Remove the fuel injectors.

    • With suitable piping fixtures, seal off the fuel rail outlets to the fuel injectors.

    • Align the piston exactly in the centre of the compression stroke using a TDC indicator tool.

    • Connect the pressure loss tester to the compressed air network (5–10 bar) and calibrate the tester.

  4. Check cylinder pressure loss.

    1. Follow the cylinder pressure loss tester tool operator manual to connect the tool to the suspicious cylinder.

    2. Check for leaks and noises.

    3. Check the cylinder pressure loss. The loss resulting from leakage is indicated in percent on the pressure gauge.

    4. If the indicated air leakage percentage is in the NOT OK area, investigate these possible root causes:

      Symptom

      Root Cause

      Noises in the exhaust

      • Exhaust valve leaking

      Noises in the intake tract

      • Inlet valve leaking

      Noises on crankcase ventilation, oil filler cap, or oil dipstick

      • Wear of cylinder track

      • Piston rings

      • Crack or hole in the piston

      • Defective cylinder head gasket

      Noises at the injector opening of the adjacent cylinder

      • Cylinder head gasket leaking

      Bubble formation in the cooling system

      • Cylinder head gasket leaking

      • Crack in the cylinder head

      Locate and correct the cause, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    5. If the indicated air leakage percentage is in the OK area, continue to the next step.

  5. The cylinder works properly.
