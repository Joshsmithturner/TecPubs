SPN-17509-31 / DTC-4465-1F

Fuel Pump Pressure has Exceeded Desired Pressure Limits

Fault Description: 

The fuel metering valve is a magnetic proportional valve (MPROP) which controls the desired fuel flow to the fuel rail. The fuel metering valve adjusts the quantity of fuel reaching the high-pressure pump based on commands received from the ECU. If actual fuel pressure exceeds requested fuel pressure, leakage will be assumed and this fault occurs.

Cause: 

The actual fuel pressure has exceeded the requested fuel pressure by 50.0 bar (725 psi).

Possible Failure Modes:

  1. Faulty fuel injectors, internal leakage or stuck open.

  2. Faulty fuel filter, clogged.

  3. Faulty low-pressure fuel system, leakage or blockage.

  4. Faulty gear pump pressure.

  5. Faulty fuel metering valve (MPROP), wiring or internal failure.

  6. Faulty fuel rail pressure sensor, wiring or internal failure.

  7. Faulty 5 bar relief valve.

Suggested Spare Parts:

  • Fuel filters

  • Fuel/water separator filter

  • 5 bar relief valve

  • Fuel metering valve (MPROP)

  • Fuel injectors

  • Tubes for fuel injectors

  • Pressure relief valve (PRV)

  • Fuel rail pressure sensor

  • Fuel pump

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Return the machine to service.

  2. Replace the fuel filter with a Tigercat filter, then remove and clean the fuel metering valve (MPROP) with compressed air.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  3. Replace the fuel metering valve (MPROP), use the PT-Box tool or LogOn to perform the Component Replacement > Replace Rail Metering Unit procedure, and then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  4. Check the status of the following related faults:

    SPN-26197-31 / DTC-6655-1F

    Fuel Metering Valve (MPROP): Short Circuit to Battery on Low Side

    SPN-30037-31 / DTC-7555-1F

    Fuel Metering Valve (MPROP): Short Circuit to Ground of the Supply Voltage

    SPN-33621-31 / DTC-8355-1F

    Fuel Metering Valve (MPROP): Over-Temperature Error

    SPN-16725 / DTC-4155

    Fuel Metering Valve (MPROP): Intermittent Electrical Connection Failure

    SPN-21077-31 / DTC-5255-1F

    Fuel Metering Valve (MPROP): Open Circuit

    SPN-5157-31 / DTC-1425-1F

    Fuel Rail Pressure Sensor: Voltage is Too High

    SPN-15461-31 / DTC-3C65-1F

    High Pressure Pump Fuel Delivery Quantity at Low Speed is Too High

    SPN-15205-31 / DTC-3B65-1F

    High-Pressure Pump Fuel Delivery Quantity in Over Run Exceeds a Maximum Threshold

    SPN-5477-31 / DTC-1565-1F

    Fuel Rail Pressure Has Exceeded Maximum Negative Deviation Limits

    SPN-8805-31 / DTC-2265-1F

    Fuel Rail Pressure Has Exceeded Maximum Positive Deviation Limits

    SPN-62533-31 / DTC-F445-1F

    Fuel Pressure Relief Valve is Open

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  5. Check for leaking fuel injectors. With the fuel pressure leak kit, install the injector backflow line and route to an appropriate container, then bring the engine up to operating temperature. At low idle, injector backflow should be 80 ml per minute maximum. Refer to Fuel Injector Leakage Test for the full procedure.

    1. If injector backflow is greater than 80 ml per minute, use the fuel rail injector plug kit to isolate each injector, one at a time. Determine which injectors are faulty and replace them.

    2. If injector backflow is less than or equal to 80 ml per minute, continue to the next step.

  6. Remove the 5 bar relief valve, clean thoroughly, and reinstall. If the fault is not resolved, replace the 5 bar relief valve with a new unit and then use the PT-Box tool or LogOn to check the status of the fault again.

    Tighten the valve to 55 N-m (41 ft-lbs). Refer to the machine's Service Manual for more information.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  7. Replace the fuel rail pressure sensor and torque to 30–40 N-m (22–30 ft-lbs), then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
