SPN-63011-31 / DTC-F623-1F

Engine Coolant Temperature Has Exceeded the Warning Threshold

Document ID: 3249 | Version: 3.2 | Last Modified Date: Sep 10, 2025

Figure 1. Coolant Temperature Sensor Connector

Position

ECU Connector-2 (Engine Harness Connector)

1

Temperature Signal

39

Ground

59



Fault Description: 

The electronic control unit (ECU) monitors the engine coolant temperature. Two engine temperature thresholds are available to signal high engine temperature. A pre-warning is set if engine temperature coolant exceeds 105.96°C (222.73°F) and a warning is set if engine coolant temperature exceeds 106.96°C (224.53°F). If the warning threshold is exceeded, this fault occurs.

Cause: 

The ECU has detected that the engine coolant temperature has exceeded the warning threshold.

Possible Failure Modes:

  1. Engine temperature too high, heavy use or high temperature operating conditions.

  2. Faulty engine cooling system, coolant low, thermostat stuck closed, restricted cooling package, or other causes.

  3. Faulty engine coolant temperature sensor, wiring or internal failure.

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 5.

  2. Check the status of the following related faults:

    SPN-4371-31 / DTC-1113-1F

    Engine Coolant Temperature Sensor: Voltage is Too High

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  3. Check the engine cooling system for an over temperature condition.

    Check the cooling system for leakage, damage, or blockage.

    Check the ambient air temperature to verify that the machine is not operating in extreme ambient temperatures.

    1. If any failures are found, repair the cooling system as necessary.

    2. If no failures are found, continue to the next step.

  4. Check the engine coolant temperature sensor wiring.

    Disconnect the ECU connector-2 (engine harness connector).

    Disconnect the engine coolant temperature sensor connector.

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    1. If the specified values are not measured, there is a short circuit condition in the engine coolant temperature sensor wiring. Locate and repair the shorted conductor.

    2. If the specified values are measured, continue to the next step.

  5. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  6. Replace the engine coolant temperature sensor, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
