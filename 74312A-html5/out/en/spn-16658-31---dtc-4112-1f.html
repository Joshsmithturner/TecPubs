SPN-16658-31 / DTC-4112-1F

Camshaft Speed Sensor: Pattern is Not Plausible

Document ID: 9063 | Version: 3.1 | Last Modified Date: Sep 6, 2024

Fault Description: 

The electronic control unit (ECU) assesses the signals received from the camshaft and crankshaft speed sensors. The angle and speed, the quality and plausibility of the signal are determined. If the ECU determines that the camshaft signal does not match the correct model (edge distance, level) or all camshafts edges required during the engine revolutions are not detected a number of times, this fault occurs. The ECU uses various monitoring strategies during engine start-up and the normal operation of the engine. If the problem is detected at engine start-up, a redundant synchronization of the angular system is carried out using only the crankshaft increment signal and the test injections to determine the engine phase. If this procedure is not carried out correctly, the injection is interrupted and the system waits for a restart.

Cause: 

The ECU has detected a camshaft signal implausibility several times.

Possible Failure Modes:

  1. Faulty phonic wheel timing.

  2. Faulty camshaft speed sensor, internal failure.

  3. Faulty camshaft speed sensor, wiring or electrical disturbances due to damaged isolation of wiring harness or special electrical features.

Suggested Spare Parts:

  • Camshaft speed sensor

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 6.

  2. Check the camshaft speed sensor signal circuit for an open circuit condition.

    Disconnect the ECU connector-2 (engine harness connector).

    Disconnect the camshaft speed sensor connector.

    With the key in the STOP position, use a multimeter to perform the following continuity check while shaking the harness to promote an intermittent connection:

    From

    To

    Value

    ECU Connector-2 (Plug) position 67

    Camshaft Speed Sensor (Plug) position 1

    There should be continuity.

    ECU Connector-2 (Plug) position 68

    Camshaft Speed Sensor (Plug) position 2

    There should be continuity.

    ECU Connector-2 (Plug) position 69

    Camshaft Speed Sensor (Plug) position 3

    There should be continuity.

    1. If there is no continuity, there is an open circuit condition in the camshaft speed sensor signal circuit. Locate and repair the broken conductor.

    2. If there is continuity, leave the connectors disconnected and continue to the next step.

  3. Check the camshaft speed sensor signal circuit for a short circuit condition.

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 68

    ECU Connector-2 (Plug) position 67

    There should be no continuity.

    ECU Connector-2 (Plug) position 68

    All other positions in ECU Connector-2 (Engine Harness Connector)

    There should be no continuity.

    ECU Connector-2 (Plug) position 67

    All other positions in ECU Connector-2 (Engine Harness Connector)

    There should be no continuity.

    1. If there is continuity, there is a short circuit condition in the camshaft speed sensor signal circuit. Locate and repair the shorted conductor.

    2. If there is no continuity, continue to the next step.

  4. Check the camshaft speed sensor signal circuit for a short circuit condition.

    With the key in the RUN position, use a multimeter to perform the following voltage check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 67

    Chassis ground

    There should be no voltage.

    ECU Connector-2 (Plug) position 68

    Chassis ground

    There should be no voltage.

    ECU Connector-2 (Plug) position 69

    Chassis ground

    There should be no voltage.

    1. If there is voltage, there is a short circuit condition in the camshaft speed sensor signal circuit. Locate and repair the shorted conductor.

    2. If there is no voltage, continue to the next step.

  5. Check the camshaft speed sensor wiring harness for any electrical interferences.

    Check if the correct camshaft sensor is installed and properly secured. The sensor should not be loose or moving.

    Optionally, to check the camshaft and crankshaft synchronization, perform the Check Camshaft and Crankshaft Synchronization procedure. This procedure requires an oscilloscope.

    1. If there is any electrical interference, an incorrect sensor is installed, the sensor is not mounted correctly, or other modifications to the sensor/harness have been made. Repair as necessary.

    2. If no faults are found, continue to the next step.

  6. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  7. With the key in the STOP position, use a multimeter to perform the following resistance check:

    From

    To

    Value

    Camshaft Speed Sensor (Receptacle) position 1

    Camshaft Speed Sensor (Receptacle) position 2

    774–946 Ω at 20°C

    1. If the measured resistance is within the specified range, continue to the next step.

    2. If the measured resistance is not within the specified range, continue to Step 9.

  8. Check phonic wheel condition and installation.

    • Check for proper timing.

    • Check for proper installation. The gap between the sensor and the phonic wheel must be within 0.3–1.8 mm for the sensor to operate correctly.

    • Check for excessive gear play.

    1. If any problems are found with the timing, installation, or gear play, repair as necessary.

    2. If no problems are found, contact the Tigercat Industries Service Department for more help.

  9. Replace the camshaft speed sensor, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
