Diagnostic Tools (PT-Box and LogOn)

Document ID: 8018 | Version: 2.11 | Last Modified Date: Oct 3, 2025

Most diagnostic functions can be performed with either the PT-Box tool and E.A.S.Y. Diagnostic Software, or with LogOn through the machine's local wifi connection.

PT-Box and E.A.S.Y. Diagnostic Software

The PT-Box is a typical engine diagnostic tool which connects to the diagnostic port on the machine. The PT-Box is the interface that allows the engine ECU to communicate with a tablet or laptop with the E.A.S.Y software installed.

The PT-Box features the following engine diagnostic tests and information:

  • Faults Memory Reading

  • Parameters Reading

  • Data Stored

  • Active Diagnosis

  • Engine Test

  • Exhaust Flap Test

  • Urea Dosing System Test (UDST)

  • Tier 4f only: Urea Dosing Leakage Check (UDLC), Urea Dosing Pump Check (UDPC), and Urea Dosing Valve Check (UDVC)

  • Component Replacement

  • Inducement Unlock / Counter Reset

Refer to the FPT Engine Troubleshooting Manual for more information.

LogOn™

LogOn is an onboard monitoring diagnostics module which comes standard on all Tigercat-TCi models. Users can connect directly to the LogOn module using any smart device or laptop with wifi capability.

LogOn for Fault Code Management

LogOn provides a list of all current fault codes along with the SPN number, component, time and date of occurrence, and troubleshooting information (if available). The search bar can also be used to quickly locate and identify specific codes.

LogOn for Engine Diagnostics

LogOn features the following engine diagnostic tests and information:

  • Engine Identifiers

  • Urea Dosing System Test (UDST)

  • Urea Dosing Leakage Check (UDLC)

  • Urea Dosing Pump Check (UDPC)

  • Urea Dosing Valve Check (UDVC)

  • Read Engine Parameters

  • Clear All Fault Codes

  • Inducement Unlock / Counter Reset (a PIN code must be obtained from the Tigercat Industries Service Department unless the user has an active RemoteLog subscription and LogOn version 4.0 or later)

  • Component Replacement (Crankcase Pressure Sensor, SCR Catalyst, NOx Upstream Sensor, NOx Downstream Sensor, Rail Metering Unit, Rail PRV, and ECU)

  • Compression Test

  • Runup Test

  • High Pressure Rail Test

Refer to the LogOn Technician's Manual (part number 66331AENG) for more information.
