SPN-42606-31 / DTC-A66E-1F

NOx Sensor Upstream (CAN): Sensor Heater Performance Insufficient

Document ID: 5001 | Version: 3.2 | Last Modified Date: Sep 10, 2025

Fault Description: 

The NOx sensor upstream of the DOC is provided with an internal monitoring system and sends information regarding the relative error state and the results of the self-diagnostics procedure to the ECU through the CAN (Controller Area Network). One of the conditions monitored is if the upstream NOx sensor reaches operating temperature within the set time. Once the dew point has been reached, the sensor has a limited time in which to heat up. If the sensor heating is not completed within this time, this fault occurs.

Cause: 

The DOC upstream NOx sensor has reported, via CAN, to the ECU that the sensor failed to reach heater working temperature within 120 seconds of reaching the dew point.

Possible Failure Modes:

  1. Faulty upstream NOx sensor, hardware or firmware.

Suggested Spare Parts:

  • NOx sensor

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 4.

  2. Check the status of the following related faults:

    SPN-19086-31 / DTC-4A8E-1F

    Upstream NOx Sensor: Supply Voltage is Out of Range

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  3. Check the correct operation of the upstream NOx sensor, following the Test the NOx Sensor procedure.

    1. If the sensor is working correctly, contact the Tigercat Industries Service Department for more help.

    2. If the sensor is not working correctly, continue to the next step.

  4. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  5. Replace the upstream NOx sensor, use the PT-Box or LogOn to perform the Component Replacement > Replace NOx Upstream Sensor procedure, and then use the PT-Box or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
