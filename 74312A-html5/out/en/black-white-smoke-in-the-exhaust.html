Black/White Smoke in the Exhaust

Document ID: 27361 | Version: 2.11 | Last Modified Date: Oct 23, 2024

  1. Record and save the following data:

    • Machine serial number

    • Machine hours

    • Clone with logs (IQAN)

    • Data stored (PT-Box)

    • Fault codes (PT-Box)

    • The vacuum reading from the filter restriction indicator before and after cleaning/checking the pre-cleaner and main filter.

  2. Verify that the turbo builds sufficient boost pressure of 117 kPa (17 psi) gauge minimum at high throttle (minimum 1,800 rpm) during operation with the engine under full load.

  3. Record the vacuum reading, then inspect and clean the air intake pre-cleaner and remove any debris/buildup surrounding it. You will need to unbolt the pre-cleaner on some machines to properly inspect and clean it at the inlet screen. The image below is an example.

    B

    Remove the Precleaner

    C

  4. Reset the air filter restriction indicator. Operate the machine for a few minutes to confirm the new vacuum reading with a clean pre-cleaner.

    A

    Filter Restriction Indicator Reset Button

    B

    Filter Restriction Indicator

    C

    Filter Restriction Threshold (25 inH20 or 6.2 kPa)

  5. Replace the primary filter element if the indicator returns to the red zone, and replace the secondary (safety) filter with every third primary filter change.

  6. Run the engine at high speed and no load, then read the air intake vacuum measurement:

    • If equipped with an air filter restriction sensor, read the air intake vacuum reading (kPa/inH2O) from the computer display.

    • If not equipped with an air filter restriction sensor, read the air intake vacuum reading (kPa/inH2O) directly from the air filter restriction indicator.

    The maximum allowable vacuum limit is 6.2 kPa (25 inH2O) with a warning set to 5.5 kPa (22 inH2O). Any values below the warning level are acceptable.

  7. Verify that the air filter O-ring is present and seated correctly. Replace if damaged or missing.

    A

    O-Ring Correctly Seated

  8. For Skidders, use IQANrun to record and save a Drive HP Test graph using the Drive/HP Test measure group. Refer to https://dealer2.tigercat.com/download/training-video-drive-horse-power-test/ for a detailed video on the test procedure.

  9. For all other machines, record and save a machine performance graph for 30 minutes with the machine in the following conditions:

    • Machine running at operating temperature

    • Machine operating under high load

    • Engine at high speed (minimum 1800 rpm)

    Use the PT-Box or LogOn to log the following parameters:

    • Engine revolutions (rpm)

    • Current injection quantity (mg/str)

    • Boost pressure (psi)

    • Exhaust gas temperature at the input of the particle trap -1

  10. Check the compressor outlet for oil build-up. If the compressor outlet shows a high level of oil build-up, send pictures to the Tigercat Industries Service Department.

  11. Check if the engine is burning oil.

  12. Check if there is coolant leakage.
