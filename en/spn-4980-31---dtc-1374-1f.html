SPN-4980-31 / DTC-1374-1F

Boost Pressure and Temperature Sensor: Temperature Voltage is Too High

Document ID: 732 | Version: 3.3 | Last Modified Date: Sep 10, 2025

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Fault Description: 

The electronic control unit (ECU) monitors the boost pressure and temperature sensor temperature signal circuit for a voltage higher than expected condition. If a voltage higher than the expected condition is detected and this fault occurs, the value will jump to a fixed temperature replacement value of 29.96°C (85.93°F).

Cause: 

The ECU has detected a voltage greater than 4.93 V for longer than 0.5 seconds in the temperature signal circuit.

Possible Failure Modes:

  1. Faulty boost pressure and temperature sensor temperature signal wire, shorted to high source or open circuit condition.

  2. Faulty boost pressure and temperature sensor, internal failure.

Suggested Spare Parts:

  • Boost pressure and temperature sensor

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 5.

  2. Check the boost pressure and temperature sensor temperature signal circuit for an open circuit condition.

    Disconnect the boost pressure and temperature sensor connector.

    Disconnect the ECU connector-2 (engine harness connector).

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 37

    Boost Pressure and Temperature Sensor (Plug) position 2

    There should be continuity.

    1. If there is no continuity, there is an open circuit condition in the temperature signal circuit. Locate and repair the broken conductor.

    2. If there is continuity, leave the connectors disconnected and continue to the next step.

  3. Check the boost pressure and temperature sensor temperature signal circuit for a short to high source condition.

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 37

    ECU Connector-2 (Plug) position 7

    There should be no continuity.

    ECU Connector-2 (Plug) position 37

    All other positions in ECU Connector-2 (Engine Harness Connector)

    There should be no continuity.

    1. If there is continuity, there is a short to high source condition in the temperature signal circuit. Locate and repair the shorted conductor.

    2. If there is no continuity, leave the connectors disconnected and continue to the next step.

  4. Check the boost pressure and temperature sensor signal circuit for a short to key power condition.

    With the key in the RUN position, use a multimeter to perform the following voltage check:

    From

    To

    Value

    ECU Connector-2 (Plug) position 37

    Chassis ground

    There should be no voltage.

    1. If there is voltage, there is a short to key power condition in the temperature signal circuit. Locate and repair the shorted conductor.

    2. If there is no voltage, continue to the next step.

  5. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  6. Replace the boost pressure and temperature sensor, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
