SPN-6218-31 / DTC-184A-1F

DEF System: Monitoring of Overpressure

Document ID: 948 | Version: 3.2 | Last Modified Date: Oct 24, 2024

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Fault Description: 

If the DEF system pressure exceeds 12.50 bar (181.25 psi) during normal operation or purging, this fault occurs.

Cause: 

The electronic control unit (ECU) has determined that the pressure in the DEF system is too high.

Possible Failure Modes:

  1. Faulty DEF system pressure or backflow line (tube), restricted.

  2. Faulty DEF system filter in supply module backflow connector, restricted.

  3. Faulty DEF system throttle in supply module backflow connector, restricted.

  4. Faulty DEF system check valve in supply module backflow connector, does not open.

  5. Faulty DEF supply module, pump defective.

  6. Faulty DEF supply module, pressure sensor defective.

Suggested Spare Parts:

  • DEF supply module

  • DEF supply module backflow connector

Required Tools:

  • One 1.5 m (5 ft) length of clear tubing, 3/8 in inside diameter, plus extra tubing as needed

  • One 1.5 m (5 ft) length of clear tubing, 1/4 in inside diameter, plus extra tubing as needed

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 8.

  2. Identify the required test tubing for the DEF supply module.

    There are two types of DEF supply module, which can be identified by the white backflow fitting.

    DEF Supply Module 91894B01 Required Test Tubing

    A

    Inlet Fitting Connection

    B

    White Backflow Fitting (Part Number 218340), 3/8 in

    1.5 m (5 ft) length of clear tubing, 3/8 in inside diameter

    DEF Supply Module 91894B02 Required Test Tubing

    A

    Inlet Fitting Connection

    B

    Orifice Backflow Fitting (Part Number 223071), 1/4 in

    1.5 m (5 ft) length of clear tubing, 1/4 in inside diameter

  3. Test the DEF supply module backflow line for a blockage.

    Example of DEF Supply Module Backflow Test

    1. Connect the appropriate clear tubing to the backflow fitting of the DEF supply module.

    2. Place the other end of the tubing in a bucket. The backflow rate and pressure are very low, so if no bucket is available, any smaller container can be used.

    3. Perform a UDST while monitoring the clear tubing. Make sure there is DEF flowing through the tubing during the test.

      1. If DEF does not flow through the tubing during the test, the backflow connector is probably blocked. Continue to the next step.

      2. If DEF flows through the tubing during the test, continue to Step 5.

  4. Check the DEF supply module backflow connector.

    Visually and tactually inspect the filter, throttle, and check valve internal to the supply module backflow connector for restrictions and damage. There should be no blockage or damage.

    Note

    Contamination in the backflow connector is difficult to see. The most effective way to rule out a blockage due to contamination is to replace the connector.

    1. If blockage or damage is found, repair or replace as needed.

    2. If no blockage or damage is found, continue to the next step.

  5. Check for blockage or damage in the DEF supply module pressure and backflow lines.

    Visually and tactually inspect the pressure and backflow lines for restrictions or damage. There should be no blockage or damage.

    1. If blockage or damage is found, repair or replace the line (tube).

    2. If no blockage or damage is found, continue to the next step.

  6. Check the status of the following related faults:

    SPN-6490-31 / DTC-195A-1F

    DEF Supply Module Pressure Sensor: Voltage is Too High

    SPN-29242-31 / DTC-723A-1F

    DEF Supply Module Pump Motor: Short Circuit to Ground

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  7. Test the DEF system using either the PT-Box or LogOn:

    • Perform the Urea Dosing System Test (UDST).

    • Perform a Urea Dosing Valve Check (UDVC) (300–360 ml).

    • Perform a Urea Dosing Pump Check (UDPC) (500–600 ml, spec 7.2).

    • Perform a Urea Dosing Leakage Check (UDLC).

    Use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  8. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates abnormal display readings, continue to the next step.

    3. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.

  9. Replace the DEF supply module, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.
