SPN-19338-31 / DTC-4B8A-1F

DEF System Unable to Build Sufficient Pressure for Dosing

Document ID: 4307 | Version: 3.3 | Last Modified Date: Oct 24, 2024

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Note

When performing multiple UDSTs with LogOn, the ignition key must be turned to STOP and then to RUN between each test.

Fault Description: 

If the DEF system pressure does not reach 6.50 bar (94.25 psi) within 60 seconds after ten pressure build up cycles, this fault occurs.

Cause: 

The electronic control unit (ECU) has determined that a low-pressure condition exists in the DEF system.

Possible Failure Modes:

  1. Faulty DEF system, in-line suction filter restricted (if equipped).

  2. Faulty DEF system, filter in inlet connector of the supply module restricted.

  3. Faulty DEF system, filter in backflow connector of the supply module restricted.

  4. Faulty DEF system, tank ventilation line restricted.

  5. Faulty DEF system, suction line frozen, blocked, or leaking (tubing, hydraulic connector, O-ring).

  6. Faulty DEF system, backflow line frozen or blocked (tubing, hydraulic connector).

  7. Faulty DEF system, leakage on suction pipe inside DEF tank unit.

  8. Faulty DEF system, supply module pump motor or pressure sensor defective.

  9. Faulty DEF system, blocked or restricted injector.

Suggested Spare Parts:

  • Supply module filter

  • Supply module

  • Complete DEF tank assembly

  • DEF lines

  • DEF injector

Required Tools:

  • 19 L (5 US gal) of new DEF in sealed containers

  • Two 1.5 m (5 ft) lengths of clear tubing, 3/8 in inside diameter, plus extra tubing as needed

  • One 1.5 m (5 ft) length of clear tubing, 1/4 in inside diameter, plus extra tubing as needed

  • Clear container

Solution:

  1. Verify that the fault is present and active and retrieve the Stored Data and Fault Codes using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Visually check the fluid level in the DEF tank and fill to an acceptable level. Air in the DEF system will cause the pressure level to decrease.

  3. Remove the DEF dosing module and inspect for any crystallization. If crystals are present at the tip of the nozzle (less than 100 micron), remove the DEF dosing module, cover the electrical connector, and soak the module in warm water for a minimum of 15 minutes. Inspect the nozzle to make sure that it is clean, then perform a UDPC. Refer to Dosing Module Soaking Procedure for the full procedure.

  4. Test the DEF dosing module to determine if the fault is present or intermittent:

    Place the DEF dosing module into a clear container and use the PT-Box tool or LogOn to perform 3 consecutive UDSTs. Save the results for each test.

    Verify that the DEF dosing module is not leaking or stuck open during the tests except for the 1 injection cycle commanded by the ECU. If the DEF dosing module is leaking, replace the DEF dosing module and perform 3 consecutive UDSTs again. Save the results for each test.

    1. If the DEF supply module passes 3 consecutive UDSTs with a non-leaking dosing module, the fault is intermittent. If the fault is intermittent, perform the Intermittent Fault Process (UDSTs Passed) procedure. The elimination process cannot be performed because there is no active failure detected by the UDST (the DEF supply module will continue to pass the UDST when components are replaced).

      The intermittent fault process includes the following checks:

      • Visually check for contamination and leaks.

      • If no contamination is found, replace the DEF supply module.

    2. If the DEF supply module fails 1 or more UDSTs with a non-leaking dosing module, the fault is present in the system and the cause of the fault can be determined through an elimination process and further testing. Perform the Elimination Process (UDSTs Failed) procedure to identify the cause of the fault.

      The elimination process includes the following checks:

      • Visually check for contamination and leaks.

      • Determine if the DEF suction hose is the cause of the fault.

      • Determine if the DEF sending unit is the cause of the fault.

      • Determine if the DEF supply module is the cause of the fault.
