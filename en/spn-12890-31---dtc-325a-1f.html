SPN-12890-31 / DTC-325A-1F

DEF Supply Module Pump Temperature is Too Low to Activate Pump

Document ID: 1929 | Version: 3.2 | Last Modified Date: Oct 24, 2024

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

1

Fluid Output Pipe of Supply Module Heating

2

DEF Return Pipe to Tank

3

Delivery Pipe to Dosing Module

4

Main Filter

5

Electrical Connector

6

Pressure Compensation Membrane

7

Fluid Input Pipe of Supply Module Heating

8

Supply Pipe from Tank

Fault Description: 

The DEF supply module pump motor control board works initially in temperature measurement mode (to determine if unfreezing/heat-up is necessary) then switches to pump actuation mode (to build up pressure and prepare to inject). If the DEF supply module pump motor control board does not switch to pump actuation mode after temperature measurement has been carried out, this fault occurs.

Cause: 

The electronic control unit (ECU) has determined that the DEF supply module pump motor control board did not switch to pump actuation mode.

Possible Failure Modes:

  1. Faulty DEF supply module, internal failure.

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Check DEF supply module operation.

    Use the PT-Box tool or LogOn to perform the Urea Dosing System Test (UDST). Follow the UDST Troubleshooting Guideline to determine the problem.

    1. If the UDST identifies a problem, follow the UDST Troubleshooting Guideline to locate and repair the failure.

    2. If the UDST does not identify a problem, remove the injector and perform a Urea Dosing Valve Check (UDVC) (300–360 ml), Urea Dosing Pump Check (UDPC) (500–600 ml), and Urea Dosing Leakage Check (UDLC).

      If no problems are found, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
