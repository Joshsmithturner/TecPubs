Fuel High Pressure Relief Test

Document ID: 4149 | Version: 2.3 | Last Modified Date: Sep 10, 2025

Caution

Fuel may be under pressure if the pressure relief valve (PRV) has malfunctioned. Use extreme caution when removing lines and performing leak checks to avoid injury.

Required Tools:

  • NEF Engine Fuel Leakage Test Kit (Part Number 212541), includes measuring cup, clear test line, and adapter to junction bolt at high-pressure pump

  • Cursor Engine Fuel Leakage Test Kit Add-On (Part Number 73470B)

A

Capped Fitting

B

Backflow Line

C

Pressure Relief Valve (PRV)

D

Clear Test Line

E

Test Line to Container

Procedure:

  1. Remove the backflow line from the PRV backflow connector.

  2. Install the capped fitting from the Leakage Test Kit to the end of the backflow line.

  3. Install the clear test line from the Leakage Test Kit to the PRV backflow connector.

  4. Place the other end of the clear test line into the large container from the Leakage Test Kit.

  5. While keeping a safe distance from the clear test line, START the engine observe the line and container for any fuel. Under normal conditions, there should be no leakage from the PRV.

    1. If there is fuel coming out of the PRV, the PRV is stuck open or leaking internally.

    2. If there is no fuel coming out of the PRV, the PRV is okay.

  6. STOP the engine, wait 10 minutes for fuel pressure to be relieved, and then reconnect the backflow line to the PRV backflow connector.
