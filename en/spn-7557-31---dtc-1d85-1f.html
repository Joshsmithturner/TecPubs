Fault Description: 

The electronic control unit (ECU) monitors fuel pressure during limp home mode. If the ECU determines that the fuel pressure has exceeded the maximum limit during limp home mode, this fault occurs.

Cause: 

The ECU has determined that the maximum fuel pressure has exceeded 1990 bar (28855 psi) for more than 1 second.

Possible Failure Modes:

  1. Faulty fuel metering valve (MPROP), wiring or internal failure.

  2. Faulty fuel injectors.

  3. Faulty fuel return lines.

  4. Faulty pressure relief valve (PRV), clogged.

  5. Faulty 5 bar relief valve, clogged.

Suggested Spare Parts:

  • Fuel injectors

  • Pressure relief valve (PRV)

  • 5 bar relief valve

  • Fuel metering valve (MPROP)

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Return the machine to service.

  2. Check the status of the following related faults:

    SPN-16725 / DTC-4155

    Fuel Metering Valve (MPROP): Intermittent Electrical Connection Failure

    SPN-21077-31 / DTC-5255-1F

    Fuel Metering Valve (MPROP): Open Circuit

    SPN-26197-31 / DTC-6655-1F

    Fuel Metering Valve (MPROP): Short Circuit to Battery on Low Side

    SPN-30037-31 / DTC-7555-1F

    Fuel Metering Valve (MPROP): Short Circuit to Ground of the Supply Voltage

    SPN-33621-31 / DTC-8355-1F

    Fuel Metering Valve (MPROP): Over-Temperature Error

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If any faults remain, continue to the next step.

  3. Check for leaking fuel injectors. With the fuel pressure leak kit, install the injector backflow line and route to an appropriate container, then bring the engine up to operating temperature. At low idle, injector backflow should be 80 ml per minute maximum. Refer to Fuel Injector Leakage Test for the full procedure.

    1. If injector backflow is greater than 80 ml per minute, use the fuel rail injector plug kit to isolate each injector, one at a time. Determine which injectors are faulty and replace them.

    2. If injector backflow is less than or equal to 80 ml per minute, continue to the next step.

  4. Check the fuel return lines for blockage. There should be no blockage.

    1. If blockage is found, repair or replace as needed, then use the PT-Box tool or LogOn to verify that the fault is no longer active.

    2. If no blockage is found, continue to the next step.

  5. Check the PRV for blockage or damage.

    1. If blockage or damage is found, replace the PRV and torque to 102 N-m (75 ft-lbs), then use the PT-Box tool or LogOn to perform the Component Replacement > Replace Rail PRV procedure and verify that the fault has been resolved.

    2. If no blockage or damage is found, reinstall the PRV, torque to 102 N-m (75 ft-lbs), and continue to the next step.

  6. Remove the 5 bar relief valve, clean thoroughly, and reinstall. If the fault is not resolved, replace the 5 bar relief valve with a new unit and then use the PT-Box tool or LogOn to check the status of the fault again.

    Tighten the valve to 55 N-m (41 ft-lbs). Refer to the machine's Service Manual for more information.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  7. Replace the fuel metering valve (MPROP), use the PT-Box tool or LogOn to perform the Component Replacement > Replace Rail Metering Unit procedure, and then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
