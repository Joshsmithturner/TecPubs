Check High-Pressure Pump Backflow Pressure

Document ID: 5217 | Version: 2.5 | Last Modified Date: Sep 10, 2025

Required Tools:

  • Fuel Circuit Pressure Check Kit (212542)

  • Cursor Engine Fuel Leakage Test Kit Add-On (Part Number 73470B)

  • Low Pressure Gauge

Procedure:

  1. Prepare to check the high-pressure pump backflow pressure.

    Location of the High-Pressure Pump Backflow

    A

    Backflow Inlet From High-Pressure Pump

    B

    Return to Main Filter (Total Backflow)

    C

    Tee Hose

    D

    Low Pressure Gauge

    1. Visually check all related components for any signs of leakage.

    2. Disconnect the high-pressure pump backflow connector.

    3. Use a tee hose to connect the high-pressure pump backflow to the disconnected backflow line.

    4. Install the low pressure gauge at the tee fitting.

    5. Tighten all connections to make sure there is no leakage.

  2. Check the high-pressure pump backflow pressure.

    1. Start the engine and let it run for several minutes.

    2. In case of non-starter: Crank the engine multiple times (maximum 10 seconds each) until a specific time period is reached.

      Notice

      To prevent damaging the starter, do not engage the starter motor for more than 10–12 seconds each time. Wait 30 seconds between each attempt to START the engine.

    3. Observe pressure build up in several engine conditions and compare the pressure values to the expected results below.

      Engine Condition

      Maximum Pressure (Absolute)

      Low speed (750 rpm)

      0.2 bar (3 psi)

      High Speed (1500 rpm)

      0.2 bar (3 psi)

  3. Check if the pressure measured is in the range explained in the table above.

    1. If the pressure values are NOT OK (backflow pressure too high), investigate these possible root causes:

      • Overflow valve in the high-pressure pump is stuck open

      • Main filter head is blocked or restricted

      Locate and correct the cause of the damage, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    2. If the pressure values are NOT OK (backflow pressure too low), investigate these possible root causes:

      • Overflow valve in the high-pressure pump is blocked, dirty, or stuck closed

      • Check tightness of tubes, hoses, and connectors for leaks and looseness

      Locate and correct the cause of the damage, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    3. If the pressure values are OK, continue to the next step.

  4. The high-pressure pump works properly.
