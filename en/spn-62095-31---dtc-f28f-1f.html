SPN-62095-31 / DTC-F28F-1F

DEF Level Too Low Warning is Active - Stage 2

Document ID: 3151 | Version: 3.2 | Last Modified Date: Feb 20, 2025

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

Note

This machine may be equipped with an FPT DEF tank, a KUS DEF tank (1st Gen), or a KUS DEF tank (2nd Gen). Check which tank is equipped before starting troubleshooting. Refer to Identifying the DEF Tank Manufacturer for more information.

Fault Description: 

The electronic control unit (ECU) monitors the level and temperature of the DEF fluid. If the level of the fluid falls below 5% of full, this fault occurs.

Cause: 

The DEF tank level and temperature sensor level signal to the ECU is less than 5% of full.

Possible Failure Modes:

  1. DEF tank fluid level, too low.

  2. Faulty DEF tank level and temperature sensor, internal failure.

  3. Faulty DEF tank level and temperature sensor, wiring.

  4. Faulty ECU software.

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Fill the DEF tank to the appropriate level, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  3. Check the DEF level sensor for proper operation.

    Remove the DEF sending unit from the DEF tank.

    Monitor the DEF level gauge while moving the level float on the DEF sending unit up and down. Check that the level float moves freely and visually check for damage.

    1. If the level float is stuck or damaged, clean, repair, or replace the DEF sending unit as required.

    2. If no faults are found or the DEF level signal does not correspond with the movement of the floater, and the machine is equipped with an FPT DEF tank, continue to the next step.

    3. If no faults are found or the DEF level signal does not correspond with the movement of the floater, and the machine is equipped with a KUS DEF tank (1st Gen or 2nd Gen), continue to Step 5.

  4. FPT DEF tanks: Check the DEF tank level and temperature sensor for internal failure.

    Disconnect the vehicle harness from the DEF tank level and temperature sensor.

    With the key in the STOP position, use a multimeter to perform the following resistance check:

    From

    To

    DEF Level and Temperature Sensor (Plug) position 1

    2. If the measured resistance is correct, contact the Tigercat Industries Service Department for more help.

  5. KUS DEF tanks: Check the DEF sending unit controller supply voltage.

    Disconnect the vehicle harness from the DEF sending unit connector.

    With the key in the RUN position, use a multimeter to perform the following voltage check on the vehicle harness side:

    From

    To

    Value

    Harness Connector position 1

    Harness Connector position 2

    1. If the voltage is present, leave the connectors disconnected and continue to the next step.

    2. If the voltage is not present, continue to Step 7.

  6. Check the condition of the DEF sending unit controller CAN circuit.

    With the key in the STOP position, use a multimeter to measure the resistance of the CAN connection on the vehicle harness side:

    From

    To

    1. If the measured resistance is not correct, continue to the next step.

    2. If the measured resistance is correct, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn

  7. Check for other aftertreatment system (ATS) CAN faults.

    Use the PT-Box tool or LogOn to determine if ATS CAN faults exist.

    1. If related ATS CAN faults are active, resolve them first and then determine if this fault is also resolved.

    2. If no related ATS CAN faults are active, the vehicle harness wiring is damaged between the harness splice and the DEF sending unit connector position 1 or 2 for supply power and ground, or position 3 or 4 for CAN signal. Locate and repair the damage to the sensor supply power and ground or CAN circuit wiring.
