Check Camshaft and Crankshaft Synchronization

Document ID: 5243 | Version: 4.3 | Last Modified Date: Sep 10, 2025

Required Tools:

  • Oscilloscope

  • Breakout harness

Preparation:

  • Check that the battery voltage is sufficient.

  • Electrical circuit is OK, like fuses, connectors, wiring harness, relays

Procedure:

  1. Visually check the camshaft sensor.

    • Visually check the sensor connector for damage or contamination.

    • Check that the sensor is mounted in the correct orientation.

    • Check if the screw of the sensor is tight.

    • Check that the sensor is correctly fitted to the engine body, with no gap between the sensor and the engine body.

    1. If the camshaft sensor is NOT OK, locate and correct the cause of the damage, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    2. If the camshaft sensor is OK, continue to the next step.

  2. Visually check the crankshaft sensor.

    • Visually check the sensor connector for damage or contamination.

    • Check that the sensor is mounted in the correct orientation.

    • Check if the screw of the sensor is tight.

    • Check that the sensor is correctly fitted to the engine body, with no gap between the sensor and the engine body.

    1. If the crankshaft sensor is NOT OK, locate and correct the cause of the damage, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    2. If the crankshaft sensor is OK, continue to the next step.

  3. Check the Camshaft Speed Sensor signal (CmSpdSens) by oscilloscope.

    1. Install a breakout harness between the engine harness camshaft sensor connector and the camshaft sensor.

    2. Use the oscilloscope to capture the CmSpdSens signal between the indicated pins.

      From

      To

      Engine Harness Camshaft Sensor Connector (Plug) position 1

      Engine Harness Camshaft Sensor Connector (Plug) position 2

    3. Keep the engine in low idle for at least 1 minute.

  4. Check the Crankshaft Speed Sensor signal (CrSpdSens) by oscilloscope.

    1. Install a breakout harness between the engine harness crankshaft sensor connector and the crankshaft sensor.

    2. Use the oscilloscope to capture the CrSpdSens signal between the indicated pins.

      From

      To

      Engine Harness Crankshaft Sensor Connector (Plug) position 1

      Engine Harness Crankshaft Sensor Connector (Plug) position 2

    3. Keep the engine in low idle for at least 1 minute.

  5. Check camshaft and crankshaft synchronization

    The camshaft and crankshaft speed sensors' outputs should have a similar waveform to the figure below:

    Camshaft and Crankshaft Sensors Output Signals Comparison (Sync in Low Idle Speed)

    A

    Crankshaft Speed Sensor Signal

    B

    Camshaft Speed Sensor Signal

    Note

    The maximum allowed output voltage peaks for the speed sensor is according to the control unit interface definition: output V minor to 100 V and major to −100 V.

    1. If the synchronization is NOT OK, investigate these possible root causes:

      • Wrong mechanical trimming

      • Sensor is loosely fitted to the engine body

      Locate and correct the cause of the damage, then use the PT-Box tool or LogOn to clear all fault codes and verify that the fault is resolved.

    2. If the synchronization is OK, continue to the next step.

  6. The camshaft and crankshaft synchronization works properly.
