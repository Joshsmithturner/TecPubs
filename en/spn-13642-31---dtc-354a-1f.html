SPN-13642-31 / DTC-354A-1F

DEF System General Pressure Check Failure

Document ID: 2066 | Version: 3.3 | Last Modified Date: Sep 10, 2025

Note

Because this fault causes inducement, it is necessary to perform the Inducement Unlock/Counter Reset once the fault has been resolved. Contact the Tigercat Industries Service Department if needed.

DEF Dosing Module

Fault Description: 

After the DEF pressure has stabilized 8.5–9.5 bar (123.2–137.8 psi), the dosing module is energized for 1 second. If the sensed pressure provided by the supply module pressure sensor exceeds 8.5 bar (123.2 psi) while the dosing module solenoid is energized (valve open) and the maximum number of heating cycles has been reached, this fault occurs. If the maximum number of heating cycles has not been reached, an additional heating cycle will be performed.

Cause: 

The electronic control unit (ECU) sensed a pressure greater than 8.5 bar (132.2 psi) while the dosing module was energized.

Possible Failure Modes:

  1. Faulty DEF system pressure lines (tubes), restricted.

  2. Faulty DEF heating system, ineffective

  3. Faulty dosing module solenoid valve, restricted or damaged.

  4. Faulty supply module, restricted or damaged.

Suggested Spare Parts:

  • DEF supply module

  • DEF dosing module

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

  2. Check for blockage or damage in the DEF system pressure lines (tube).

    Visually and tactually inspect the system pressure lines for restrictions or damage. There should be no blockage or damage.

    1. If blockage or damage is found, repair or replace the line (tube).

    2. If no blockage or damage is found, continue to the next step.

  3. Check the DEF system pressure lines (tubes) for freezing.

    Visually and tactually inspect the system pressure lines for blockage due to freezing. There should be no blockage.

    1. If any blockage due to freezing is found, the DEF heating system is deficient. Locate the deficiency and and repair or replace as required.

    2. If no blockage due to freezing is found, continue to the next step.

  4. Test the DEF system using either the PT-Box or LogOn:

    • Perform the Urea Dosing System Test (UDST).

    • Perform a Urea Dosing Valve Check (UDVC) (300–360 ml).

    • Perform a Urea Dosing Pump Check (UDPC) (500–600 ml, spec 7.2).

    • Perform a Urea Dosing Leakage Check (UDLC).

    Use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  5. Check the status of the following related faults:

    SPN-14202-31 / DTC-377A-1F

    DEF Dosing Module is Blocked Closed

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  6. Replace the DEF dosing module, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  7. Replace the DEF supply module, then use the PT-Box tool or LogOn to verify that the fault has been resolved.

    1. If the fault has been resolved, use the PT-Box tool or LogOn to perform the Inducement Unlock/Counter Reset, then return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.

      Please have the following information ready:

      • Machine serial number

      • Machine hours

      • IQAN clone

      • FPT fault codes and stored data from the PT-Box or LogOn
