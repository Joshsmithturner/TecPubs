SPN-34714-31 / DTC-879A-1F

Coolant Control Valve: Over-Temperature Error

Document ID: 10202 | Version: 3.1 | Last Modified Date: Oct 29, 2024

Fault Description: 

The electronic control unit (ECU) monitors the circuit of the coolant control valve to check for an over-temperature/current condition. If the ECU detects an over-temperature/current condition, this fault occurs.

Cause: 

The ECU has detected an over-temperature/current condition in the coolant control valve circuit.

Possible Failure Modes:

  1. Faulty coolant control valve, internal failure.

  2. Faulty coolant control valve, wiring harness.

Suggested Spare Parts:

  • Coolant control valve

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 5.

  2. Check the coolant control valve for an internal fault.

    Disconnect the coolant control valve.

    Use a multimeter to perform the following resistance check:

    From

    To

    Value

    Coolant Control Valve (Receptacle) position 1

    Coolant Control Valve (Receptacle) position 3

    29–35 Ω

    1. If the value is not within the specified range, the coolant control valve has failed internally. Replace the coolant control valve.

    2. If the value is within the specified range, continue to the next step.

  3. Check the coolant control valve circuit for a short circuit to battery power condition.

    Disconnect the ECU connector-1 (machine harness connector).

    With the key in the STOP position, use a multimeter to perform the following continuity check:

    From

    To

    Value

    ECU Connector-1 (Plug) position 72

    ECU Connector-1 (Plug) position 50

    There should be no continuity.

    ECU Connector-1 (Plug) position 72

    All other positions in ECU Connector-1 (Machine Harness Connector)

    There should be no continuity.

    1. If there is continuity, there is a short circuit condition in the coolant control valve circuit. Locate and repair the damaged conductor.

    2. If there is no continuity, continue to the next step.

  4. Check the coolant control valve circuit for a short circuit to key power condition.

    Disconnect the ECU connector-1 (machine harness connector).

    With the key in the RUN position, use a multimeter to perform the following voltage check:

    From

    To

    Value

    ECU Connector-1 (Plug) position 72

    Chassis ground

    There should be no voltage.

    1. If there is voltage, there is a short circuit condition in the coolant control valve circuit. Locate and repair the shorted conductor.

    2. If there is no voltage, contact the Tigercat Industries Service Department for more help.

  5. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.
