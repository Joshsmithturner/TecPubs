Replacing the DEF Supply Module Filter

Document ID: 13156 | Version: 2.3 | Last Modified Date: Sep 10, 2025

Notice

Clean the area around the DEF supply module ports and filter cover before removing the filter. Removing the filter without cleaning the surrounding area can result in DEF contamination, which could damage the aftertreatment system.

  1. Clean the area around the DEF dosing module filter, especially the inlet port and filter cover.

  2. Place a container below the DEF supply module to catch any spilled DEF.

  3. Use a 27 mm (1 ¹⁄₁₆ in) wrench to turn the filter cover counterclockwise and remove the cover.

  4. Remove the filter element from the supply module and note the colour of the filter inside (black or grey).

  5. Prepare the filter removal tool. If the filter is black, the BLACK (B) side of the tool should be inserted into the filter; if the filter is grey, the (G) GREY side of the tool should be inserted into the filter.

  6. Insert the filter removal tool completely into the filter until a click is felt or heard, indicating the correct position.

  7. Pull the filter removal tool straight out from the DEF supply module to remove the filter from its seat. If required, insert a tool through the slot of the filter removal tool to assist with pulling the tool.

    Notice

    When the filter is removed, protect the filter seat to ensure that no particles or contamination are introduced into the DEF supply module.

  8. Clean the DEF supply module surfaces and the filter cover with distilled or demineralized water.

  9. Check the area around the filter for cracks. If cracks are found, replace the DEF supply module.

  10. Lubricate the O-rings on the new filter with clean DEF.

  11. Install the new filter.

  12. Install the new filter element.

  13. Use a 27 mm (1 ¹⁄₁₆ in) wrench to install the filter cover and torque to 20 N-m (15 lb-ft).

  14. Install the DEF supply module on the machine, then check for leaks.
