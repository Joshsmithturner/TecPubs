Test the Exhaust Gas Temperature Sensor

Document ID: 12640 | Version: 2.2 | Last Modified Date: Sep 10, 2025

Description:

The exhaust gas temperature sensors are components of the aftertreatment system and measures the temperature at the inlet of the DOC, the inlet of the SCR, and the outlet of the SCR. Each sensor consists of an internal platinum resistor crossed by a voltage of 5 V, which allows indirect measurement of the resistance as the temperature varies. The sensor allows the electronic control unit (ECU) to constantly adapt the supply of DEF to the engine load conditions.

To refer to the technical data relating to the exhaust gas temperature sensors, see Technical Data for Exhaust Gas Temperature Sensor.

Note

The error codes relating to the downstream SCR exhaust gas temperature sensor and the upstream SCR exhaust gas temperature sensor immediately become inactive when the SCR outlet or inlet temperature drops below the threshold value. With the vehicle in the workshop, it is likely that the error code will be inactive.

Test:

  1. Check that the temperature sensors on the aftertreatment system are installed correctly and in the specified positions.

  2. Use the PT-Box tool or LogOn (Version 3.2 or later) to check that when the engine is cold, the values of the temperature sensors on the aftertreatment system are all aligned and coincide with the ambient temperature.

    The sensors to be checked are the following:

    1. Upstream DOC exhaust gas temperature sensor

    2. Downstream DOC exhaust gas temperature sensor (Stage V only)

    3. Upstream SCR exhaust gas temperature sensor

    4. Downstream SCR exhaust gas temperature sensor

    Any misalignment of the temperature value can indicate a fault in a sensor or in the relative electrical circuit.

    1. If the temperature values corresponding to the sensors detected are in-line, continue to the next step.

    2. If the temperature values relating to the sensors in question are not aligned, identify the sensor with the fault and continue to Step 6.

  3. Make sure the engine is operating at the correct temperature by performing the following steps:

    1. Keep the ignition switch in the RUN position for 15 seconds.

    2. Start the engine and maintain idle speed for 15 seconds.

    3. Keep the engine running at 2000 rpm for 30 seconds.

    4. Keep the engine running at 1600 rpm for 30 seconds with the exhaust flap closed at 90%. The exhaust flap position can be set using the PT-Box tool or LogOn.

    5. Keep the engine running at idle speed and continue to the next step.

  4. Use the PT-Box tool or LogOn to check that the exhaust gas temperature sensors show a decreasing temperature, starting from the sensor closest to the engine and moving towards the one furthest away in the exhaust.

    The test should last approximately 300 seconds with a sampling frequency of 200 milliseconds.

    The sensors to be checked with a decreasing temperature sequence are:

    1. Upstream DOC exhaust gas temperature sensor

    2. Downstream DOC exhaust gas temperature sensor (Stage V only)

    3. Upstream SCR exhaust gas temperature sensor

    4. Downstream SCR exhaust gas temperature sensor

    1. If the temperature values relating to the sensors in question do observe the decreasing temperature trend, STOP the engine and continue to the next step.

    2. If the temperature values relating to the sensors in question do not observe the decreasing temperature trend, this indicates a malfunction in a sensor or the relative electrical circuit. STOP the engine and continue to Step 6.

  5. Check that the sensors in question return appropriate resistance values.

    Warning

    This procedure involves handling very hot parts. Before starting, make sure you are wearing suitable PPE (gloves, shoes, glasses, overalls). Failure to comply with these prescriptions can result in the risk of serious injury from contact with hot surfaces.

    1. Disconnect the sensor from the wiring harness.

    2. Remove the sensor from the aftertreatment system.

    3. Place the sensor in a pipe which is of the same diameter and material as the one on the vehicle.

    4. Heat the sensor using a digital hot air blower.

    5. Use a thermal probe to check the temperature.

    6. Use a precision multimeter to read the resistance values between positions 1 and 2 of the temperature sensor in question and compare them with those listed in the table below.
