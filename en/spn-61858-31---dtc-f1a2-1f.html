SPN-61858-31 / DTC-F1A2-1F

Engine Overspeed: Overspeed Condition Identified

Document ID: 7514 | Version: 2.3 | Last Modified Date: Sep 10, 2025

Fault Description: 

The electronic control unit (ECU) monitors engine speed to verify the presence of an overspeed condition. If engine overspeed is detected, this fault occurs.

An engine overspeed condition can be caused by the operational status, for example driving downhill; in this case, the fault does not indicate an error but simply informs about a misuse of the engine.

Cause: 

The ECU has detected an engine overspeed condition.

Possible Failure Modes:

  1. Operating error (travelling downhill).

  2. Engine out of control. Engine continues to accelerate without any operator input (fuel or oil leak into cylinders).

  3. Faulty fuel injectors, leak into cylinders (causing engine runaway condition).

  4. Faulty camshaft speed sensor, wiring or internal failure.

  5. Faulty crankshaft speed sensor, wiring or internal failure.

Solution:

  1. Verify that the fault is present and active using either the PT-Box tool or LogOn, then retrieve the IQAN clone.

    1. If the fault is present and active, continue to the next step.

    2. If the fault is no longer present or in an inactive state, the fault may be intermittent. Continue to Step 5.

  2. Check the status of the following related faults:

    SPN-13074-31 / DTC-3312-1F

    Camshaft and Crankshaft Speed Sensor Values are Not Plausible Compared to One Another

    SPN-16658-31 / DTC-4112-1F

    Camshaft Speed Sensor: Pattern is Not Plausible

    SPN-16914-31 / DTC-4212-1F

    Camshaft Speed Sensor: Values are Not Plausible

    SPN-17426-31 / DTC-4412-1F

    Crankshaft Speed Sensor: Pattern is Not Plausible

    SPN-17682-31 / DTC-4512-1F

    Crankshaft Speed Sensor: Values Are Not Plausible

    1. If any of the listed faults are active, resolve them first and then determine if this fault is also resolved.

    2. If none of the listed faults are active, continue to the next step.

  3. Verify that this fault was not caused by a downhill travel condition or any other operating conditions that may have caused this fault.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, continue to the next step.

  4. Perform the following procedures:

    • Check the engine oil level.

    • Check Camshaft and Crankshaft Synchronization

    • Cylinder Compression Test

    • Fuel Injector Leakage Test

    Correct any damage identified by the procedures, then verify that the fault is resolved.

    1. If the fault has been resolved, return the machine to service.

    2. If the fault is not resolved, contact the Tigercat Industries Service Department for more help.

  5. Visually inspect the relevant harnesses and connectors for bent or dislocated pins, corroded terminals, broken wires, and signs of heat damage or pinch damage to the harness braiding. Verify that the connectors are fully installed. Flex the harnesses involved to reveal intermittent breaks or shorts in the wiring. Operate the machine while you monitor the display.

    1. If you find damage or the display indicates abnormal display readings, repair the damage discovered during the inspection or locate and repair the abnormal display condition, then verify that the fault has been resolved.

    2. If you do not find damage and the display indicates only normal readings, delete the fault code and return the machine to service.
