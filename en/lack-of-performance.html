Lack of Performance

Document ID: 27359 | Version: 2.12 | Last Modified Date: Oct 25, 2024

  1. Record and save the following data:

    • Machine serial number

    • Machine hours

    • Clone with logs (IQAN)

    • Data stored (PT-Box)

    • Fault codes (PT-Box)

    • The vacuum reading from the filter restriction indicator before and after cleaning/checking the pre-cleaner and main filter.

  2. Record the vacuum reading, then inspect and clean the air intake pre-cleaner and remove any debris/buildup surrounding it. You will need to unbolt the pre-cleaner on some machines to properly inspect and clean it at the inlet screen. The image below is an example.

  4. Replace the primary filter element if the indicator returns to the red zone, and replace the secondary (safety) filter with every third primary filter change.

  5. Run the engine at high speed and no load, then read the air intake vacuum measurement:

    • If equipped with an air filter restriction sensor, read the air intake vacuum reading (kPa/inH2O) from the computer display.

    • If not equipped with an air filter restriction sensor, read the air intake vacuum reading (kPa/inH2O) directly from the air filter restriction indicator.

    The maximum allowable vacuum limit is 6.2 kPa (25 inH2O) with a warning set to 5.5 kPa (22 inH2O). Any values below the warning level are acceptable.

  6. Verify that the air filter O-ring is present and seated correctly. Replace if damaged or missing.

    A

    O-Ring Correctly Seated

  7. Squeeze the air cleaner unloader valve to remove any debris/dust buildup. In a dusty environment, clear this valve every 2 hours.

  8. Replace the crankcase ventilation (CCV) filter if the previous interval is greater than 500 hours.

  9. Inspect charge air cooler (CAC) pipe connection bellows for leaks and loose clamps. Release the clamp, loosen and reseat the bellow onto the pipe, and re-tighten the clamp if a leak is suspected. Spare nuts are available for the BH777 clamps (part number 221985).

  10. Remove the turbo compressor inlet and outlet connections and inspect for damaged wheel fins, broken shaft, or oil coking buildup. If any of these observations are noted, take a clear picture or video of the affected area and send it to the Tigercat Industries Service Department.

  11. Remove the turbo turbine (exhaust) outlet connection and inspect for oil or broken turbine fins. If any of these observations are noted, take a clear picture or video of the affected area and send it to the Tigercat Industries Service Department.

    • Machine running at operating temperature

    • Machine operating under high load

    • Engine at high speed (minimum 1800 rpm)

    2. Validate the boost pressure value by connecting a manual pressure gauge.

  13. For Skidders, use IQANrun to record and save a Drive HP Test graph using the Drive/HP Test measure group. Refer to https://dealer2.tigercat.com/download/training-video-drive-horse-power-test/ for a detailed video on the test procedure.

  14. For all other machines, use IQANrun to record and save a machine performance graph for 5 minutes using the J1939 Parameter Inputs measure group, with the machine in the following conditions:

    • Machine running at operating temperature

    • Machine operating under high load

    • Engine at high speed (minimum 1800 rpm)

    1. With the machine operating in the required conditions, click the Play button to start recording.

    2. After 5 minutes (300 seconds), click the Stop button to stop recording.

    3. Click File > Save to save the machine performance graph.

  15. Complete the checklist below and send all requested information to the Tigercat Industries Service Department.

    Lack of Performance Checklist

    • Record the current boost pressure.

    • Inspect and clean the pre-cleaner.

    • Reset the filter restriction indicator, operate the machine for a few minutes, and then record the new air filter restriction indicator reading.

    • Replace the engine air filter elements if necessary.

    • Verify that the air filter O-ring is present and seated correctly.

    • Replace the CCV filter.

    • Inspect for and fix leaks and loose clamps on the charge air cooler pipe bellows.

    • Inspect and send pictures of the turbo compressor inlet/outlet if any of the following are detected:

      • Compressor wheel fin damage

      • Broken turbo shaft

      • Oil coking present

    • Inspect and send pictures of the turbo turbine outlet if any of the following are detected:

      • Turbine wheel fin damage

      • Oil present

    • Record and save the following data:

      • Machine Serial Number

      • Machine Hours

      • Clone with logs (IQAN)

      • Data stored (PT-Box)

      • Fault Codes (PT-Box)

    • Record and save the machine performance graph.
